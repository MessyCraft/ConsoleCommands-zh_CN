# ConsoleCommands

命令:

/executeconsole: 以控制台身份执行命令或发送消息

    用法: /executeconsole <command>

/executeplayer: 以玩家身份执行命令

    别名: /execp,/fsudo
    用法: /executeplayer <玩家> c:<消息>/<命令>
    
/loopcmd: 循环执行命令

    用法: /loopcmd <次数> <命令>

/ccreload: 重载配置
    
    用法: /ccreload
